## Değişken tanımlamam

`x = 5` : x değişkenine 5 değerini atar.

_Değişken değerleri daha sonra değiştirilebilir._

```python
x = 5

print(x) # 5

x = 10

print(x) # 10
```

## Matematiksel Operatörler

`+` : Toplama

`-` : Çıkarma

`*` : Çarpma

`/` : Bölme

`**` : Üs alma

`%` : Mod alma

## Mantıksal Operatörler

`==` : Eşitlik

`!=` : Eşit değil

`>` : Büyüktür

`<` : Küçüktür

`>=` : Büyük eşittir

`<=` : Küçük eşittir

`and` : Ve

`or` : Veya

## Karar Yapıları

`if` : Eğer

`elif` : Eğer değilse

`else` : Değilse
